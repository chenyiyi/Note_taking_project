# Project Title

Note-taking project allows user to login in to manage their daily notes.

## Getting Started

After you succeessfully set up the Angular 2, NodeJS and mongoDB, you can run this project on the local machine.
First you need to run the mongoDB, find the path for mongoDB, cd into bin folder, run "./mongod" command to run mongoDB database. To see the collections in database, you can run "./mongo" command.
Second, if you are using Visual Studio or other framework to run the project, open the terminal inside the IDE, run "npm run build" command, then open another terminal and run "npm start" command.
You may want to install npm first.
The path for the this project is "http://localhost:3000/"

### Prerequisites

This project has not been deployed yet. To run this project, you may want to set up the newest version of Angular(2.0), NodeJS and mongoDB.
